## 核心概念

Nuwa 围绕以下几个核心概念构建：

### 节点架构 (Node Architecture)
Nuwa 中的所有组件都是 `Node`，可以链接在一起创建复杂的处理流水线。

### ReAct 模式 (ReAct Pattern)
ReAct（推理 + 行动）模式允许 Agent 逐步思考并在需要时使用工具。

### 工具集成 (Tool Integration)
通过基于装饰器的注册方式，轻松注册和使用自定义工具。

### 消息管理 (Message Management)
内置对话历史支持，可插拔的存储后端。

## 主要特性

- **流式响应支持**：实时生成响应
- **MCP 集成**：连接到模型上下文协议 (Model Context Protocol) 端点
- **自定义系统提示**：动态提示模板
- **时间感知**：自动包含时间戳
- **多轮对话**：持久化聊天历史
- **错误处理**：健壮的 JSON 解析和错误恢复
